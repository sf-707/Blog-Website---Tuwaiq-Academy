## About This Project
This is a simple, fully functional blog website I built using Django. It was developed as a backend project for the **Tuwaiq Academy** bootcamp. 

The main focus of this project was to apply Django's MVT architecture, handle database relations (SQLite), and also i build a clean interface from scratch.

## What's Inside?
* **Custom Dark Theme:** A fully responsive UI styled completely with native CSS (Grid & Flexbox) and the 'Tajawal' Arabic font.
* **Database Relations:** Connected tables for `Users`, `Categories`, `Posts`, and `Comments` using Foreign Keys.
* **Django Admin:** Pre-configured to easily manage the blog's content.
* **Dynamic Pages:** Everything is connected smoothly—from browsing all articles to reading specific post details and checking user comments.
